# 🏛️ Taupe Consulting: Advanced AI Orchestration

Welcome to the central repository for **Taupe Consulting**. This project serves as the professional portfolio, front-end, and architectural blueprint for production-grade AI ecosystems designed for high-trust environments, specialized enterprise solutions, and emerging markets.

## 🚀 The Mission
Taupe Consulting bridges the gap between raw Large Language Model (LLM) power and localized, evidence-based strategic insights. We specialize in **Retrieval-Augmented Generation (RAG)**, custom dataset crowdsourcing, automated validation pipelines, and brand-specific persona engineering.

## 🏗️ The Taupe AI Core Architecture

Our proprietary framework ensures that every AI response is fast, stylistically aligned, and factually verifiable.

### **Tier 1: The Intelligence Layer (The Engine)**
* **Core Model:** Gemini 3 Flash.
* **Optimization:** Strict hyperparameter tuning (Temperature: 0.1–0.3) and Top-p sampling to eliminate hallucinations.
* **Capability:** High-speed synthesis of massive datasets via Google Generative AI API.

### **Tier 2: The Brand DNA (The Persona)**
* **Role:** The "Executive Filter."
* **Localization:** Specialized system instructions for West African markets and enterprise-specific brand alignment.
* **Output:** Forces the AI into a **Top-Tier Strategic Consultant** voice, utilizing executive formatting (Markdown tables, executive summaries).

### **Tier 3: The Knowledge Vault (The Memory)**
* **Database:** Weaviate (Vector Database).
* **Advanced RAG:** * **Parsing:** Unstructured data handling for PDF/DOCX.
    * **Smart Chunking:** Recursive Character Text Splitting ($1200$ chunk size, $150$ overlap).
    * **Transparency:** Metadata injection for source tracking and page-number citations.

## 🌍 Featured Edge Cases: Indigenous Language Preservation & Audio Pipelines

As part of our commitment to specialized data modernization and edge-case AI applications, we build end-to-end pipelines for low-resource language preservation. Our current showcase focuses on the **Tagbana (Tagouana)** language via automated data collection and peer-validation frameworks.

### 🎙️ 1. Tagbana Speech Collector
A mobile-optimized, high-performance crowdsourcing web interface built to ingest clean local audio data directly from native speakers over standard messaging networks (e.g., WhatsApp).
* **Engineering Highlights:** Fixed sample-rate audio rendering (16kHz mono) tailored specifically for downstream Automatic Speech Recognition (ASR) training (e.g., fine-tuning Whisper/Wav2Vec2 models).
* **UX Optimization:** Built-in mobile browser initialization fixes ensuring instant native hardware microphone access.

### ⚖️ 2. Tagbana Speech Validator
A distributed peer-review interface where native speakers vote on, clean, and validate submitted raw audio datasets before they enter our secure AI data warehouse.
* **Engineering Highlights:** Secure repository integration, automated consensus thresholds, and real-time state updates to guarantee dataset high fidelity.

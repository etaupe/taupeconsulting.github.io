# 🏛️ Taupe Consulting: Advanced AI Orchestration

Welcome to the central repository for **Taupe Consulting**. This project serves as the professional front-end and architectural blueprint for a production-grade, Three-Tier AI ecosystem designed for high-trust environments and emerging markets.

## 🚀 The Mission
Taupe Consulting bridges the gap between raw Large Language Model (LLM) power and localized, evidence-based strategic insights. We specialize in **Retrieval-Augmented Generation (RAG)**, modular Python development, and brand-specific persona engineering.

---

## 🏗️ The Taupe AI Three-Tier Architecture

Our proprietary framework ensures that every AI response is fast, stylistically aligned, and factually verifiable.

### **Tier 1: The Intelligence Layer (The Engine)**
* **Core Model:** Gemini 3 Flash.
* **Optimization:** Strict hyperparameter tuning (Temperature: 0.1–0.3) and Top-p sampling to eliminate hallucinations.
* **Capability:** High-speed synthesis of massive datasets via Google Generative AI API.

### **Tier 2: The Brand DNA (The Persona)**
* **Role:** The "Executive Filter."
* **Localization:** Specialized system instructions for West African markets.
* **Output:** Forces the AI into a **Top-Tier Strategic Consultant** voice, utilizing executive formatting (Markdown tables, executive summaries).

### **Tier 3: The Knowledge Vault (The Memory)**
* **Database:** Weaviate (Vector Database).
* **Advanced RAG:** * **Parsing:** Unstructured data handling for PDF/DOCX.
    * **Smart Chunking:** Recursive Character Text Splitting ($1200$ chunk size, $150$ overlap).
    * **Transparency:** Metadata injection for source tracking and page-number citations.
